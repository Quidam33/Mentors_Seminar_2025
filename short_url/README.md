# ShortURL Service

Проект сервиса для формирования коротких ссылок  

## Описание 
Этот проект представляет собой реализацию RESTful API для создания коротких ссылок. 
Приложение реализует функционал создания короткого URL, редиректа пользователя на оригинальный полный URL и предоставления информации о ссылке.

## Возможности и эндпоинты

- POST /shorten - создать короткую ссылку
- GET /{short_id} - вернуть оригинальную ссылку 
- GET /stats/{short_id} - получить информацию о ссылке
