# TODO Service

Проект ToDo-сервиса для создания и управления задачами.

## Описание 
Этот проект представляет собой реализацию RESTful API для управления списком задач. Приложение реализует функционал создания задач, обновления и удаления, а также получение полного перечня задач пользователя.

## Возможности и эндпоинты

- POST /items - создать новую задачу
- GET /items - получить список всех задач
- GET /items/{item_id} - получить задачу по id
- PUT /items/{item_id} - обновить информацию в задаче (статус, название, описание)
- DELETE /items/{item_id} - удалить задачу
